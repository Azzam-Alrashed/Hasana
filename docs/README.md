# Hasana Product Docs

This folder contains product and requirements documents for Hasana, the daily operating system for Muslim professionals.

## Current Documents

- [2026-05-26 Hasana Product SRS](2026-05-26-hasana-product-srs.md): product agent source of truth covering vision, personas, SRS, requirements, user flows, priorities, acceptance criteria, and open questions.
- [2026-05-25 Hasana Gamified Garden Requirements - English](2026-05-25-hasana-gamified-garden-requirements-en.md): idea-phase requirements for the lifelong worship garden.
- [2026-05-25 Hasana Gamified Garden Requirements - Arabic](2026-05-25-hasana-gamified-garden-requirements-ar.md): Arabic version of the idea-phase garden requirements.
- [Hasana App Store Readiness](app-store-readiness.md): checklist of requirements, metadata drafts, and manual QA items before submission.

## Status Convention

- Idea-phase: product direction and unresolved decisions.
- Draft: actionable enough for design and implementation planning, but still open to review.
- Approved: ready to drive sprint scope and acceptance criteria.
