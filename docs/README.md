# Hasana Product Docs

This folder contains product and requirements documents for Hasana, a garden-first worship consistency app for young Muslims.

The current MVP centers on a private lifelong garden for Fajr, Dhuhr, Asr, Maghrib, Isha, Quran, Adhkar, and Witr. Core worship logging is local-only, account-free, and does not require network access. The support/payment surface is disabled for MVP and must not process payments or collect payment information.

Product language should stay gentle in Arabic and English. These docs avoid medical claims, religious authority claims, privacy claims beyond the implemented local-only model, and payment claims beyond the disabled placeholder.

## Status Convention

- Idea-phase: product direction and unresolved decisions.
- Draft: actionable enough for design and implementation planning, but still open to review.
- Approved: ready to drive sprint scope and acceptance criteria.
